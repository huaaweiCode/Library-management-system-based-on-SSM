# 基于SSM 的图书馆管理系统3

#### Description
基于SSM 的图书馆管理系统拥有两个角色，分别为管理员和学生。具体功能如下：

**管理员**：查看、添加图书，查看添加学生，借还管理，修改密码等

**学生**：查加检索图书、查看个人信息、查看个人借阅情况、修改密码
